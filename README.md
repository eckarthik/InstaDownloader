# Instagram Posts Downloader  [![Netlify Status](https://api.netlify.com/api/v1/badges/147dc820-e1a3-40d4-b270-28fbd2ae162d/deploy-status)](https://app.netlify.com/sites/eckarthik-instadl/deploys)

A React based website to download the contents from an Instagram profile.

# Features!
1. Download Pictures/Videos from a post
2. Download HD Profile Picture
3. Download IGTV Videos
4. Search the posts by a hashtag and download them
5. Ability to play the video before downloading
